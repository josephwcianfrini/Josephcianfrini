## Usage

1. Open `index.html` in a modern web browser
2. Enter a business website URL in the input field
3. Click "Analyze" to start the SEO analysis
4. View comprehensive results across all SEO pillars
5. Navigate through tabs to see detailed breakdowns
6. Check the "History" section to review past analyses

## Key Metrics Analyzed

- **Meta Tags**: Title, description, Open Graph, canonical tags
- **Keywords**: Frequency, density, distribution
- **Backlinks**: Total count, referring domains, quality score
- **Site Structure**: Sitemap, robots.txt, structured data
- **Performance**: Page load time, Core Web Vitals
